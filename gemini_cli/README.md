# Gemini CLI Install Guides

The [Gemini CLI](https://github.com/modelcontextprotocol/servers/tree/main/src/gemini) is a command-line interface for interacting with Google's Gemini models. It provides an interactive TUI for AI-assisted development, code review, and complex task automation.

This directory contains guides for configuring and optimizing the Gemini CLI.

## Available Guides
- [Reducing Thinking Level](./reduce_thinking_level.md): How to minimize the reasoning depth for faster responses.
