# OpenCode Knowledge Library

This repository is a practical, organized collection of guides, scripts, and reference materials for setting up and mastering **OpenCode**.

Rather than being a simple installation manual, this folder serves as a living knowledge base for everything related to the OpenCode ecosystem.

## 📂 Repository Structure

The documentation is organized into two logical blocks to make navigation intuitive:

### 🛠️ Block 1: Setup & Installation
*Focuses on getting the environment ready and the software running.*

- **`01_...` to `03_...`**: Shell scripts for automated environment creation (Conda) and OpenCode installation.
- **`04_getting_started_guide.md`**: A comprehensive walkthrough from Docker/vLLM setup to your first session.

### 🧠 Block 2: Reference & Mastery
*Focuses on advanced usage, customization, and extending capabilities.*

- **`05_skills_guide.md`**: High-level guide on using and installing **Skills** to extend model reasoning and tool use.
- **`06_cheat_list.md`**: A quick-reference sheet for CLI commands, TUI slash commands, and keyboard shortcuts.
- **`07_github_issues.md`**: Guidance on interacting with the project via GitHub.
- **`08_opencode_config_template.md`**: A template for managing complex multi-provider configurations.

---

## 🚀 Quick Start

If you are new here, start with [**04_getting_started_guide.md**](./04_getting_started_guide.md) to set up your local environment and launch your first session.

## 🛠️ Extending OpenCode

Once you are up and running, we highly recommend exploring **Skills**. They allow you to transform OpenCode from a standard chat interface into a powerful agent capable of specialized tasks like advanced PDF parsing or complex coding workflows. See the [**Skills Guide**](./05_skills_guide.md) for more.
